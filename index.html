No te preocupes, Aquí tienes una pequeña ayuda!

Estás a punto de aprender los selectores CSS! Los selectores son como seleccionas un elemento y le aplicas estilo.

Ejemplo 1 - Una regla CSS

p {
   margin-bottom: 12px;
}

Aquí, la "p" es el selector (selecciona todos los elementos <p>) y les aplica el estilo margin-bottom.

Para jugar, escribe el selector CSS en el editor debajo para elegir el objeto en la mesa. Si lo haces bien, pasaras al próximo nivel.

Pasa el mouse sobre el objeto en la mesa para ver su etiqueta HTML.

A la derecha tienes la ayuda del nivel! →
